# Daily Vocabulary(Java)

A simple Java console application that fetches a random English word using a public API. This project is beginner-friendly example of using HTTP APIs in Java.

---

## Features

- Fetches a **random word** from an online API  
- Uses **Java HttpURLConnection**
- Simple and clean **console output**
- Beginner-friendly project

---

## Technologies Used

- **Java (JDK 8 or above)**
- **HttpURLConnection**
- **BufferedReader & InputStreamReader**
